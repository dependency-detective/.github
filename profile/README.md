# Dependency Detective

> Investigating software dependencies and supply chains.

## Mission

Dependency Detective investigates dependency ecosystems.

Dependencies are useful.

Unnecessary dependencies are liabilities.

## Jurisdiction

Investigations include:

- Dependency Sprawl
- Supply Chain Risk
- License Risk
- Duplicate Capabilities
- Abandoned Libraries

## Core Question

> What dependency should not exist?

## Relationship to DOS

Dependency Detective is a specialization of the Detective Operating System.

## Philosophy

Every dependency should justify its existence.
